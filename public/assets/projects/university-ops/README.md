# University Ops Image Pack

Drop your final visuals in this folder to match the Behance-style template.

## Required filenames

- `hero-showcase.png`
- `module-home.png`
- `module-analytics.png`
- `module-management.png`
- `module-communication.png`
- `module-community.png`
- `module-settings.png`

## Recommended specs

- Aspect ratio: `16:10` or `16:9`
- Minimum width: `1800px` for `hero-showcase.png`
- Minimum width: `1400px` for each `module-*.png`
- Format: `.png` (or convert in-code to `.webp` if needed)
- Export quality: crisp text at 100% zoom on desktop

## Notes

- The page auto-loads these files.
- If a file is missing, the layout falls back to the built-in interactive preview.
